# Sales Performance Analytics Dashboard

## Business Problem
Business stakeholders lacked a consolidated view of sales, profitability, and performance trends across regions, categories, and products.

## Objective
To build an interactive Power BI dashboard that provides a high-level performance overview and enables deeper analysis through filters and drill-downs.

## Dataset
- Retail Superstore sales dataset
- Contains order-level transaction data including sales, profit, region, category, and product details

## Approach
- Cleaned and prepared data using Power Query
- Standardized data types and column naming
- Built KPI cards for executive overview
- Created trend and category-level visuals
- Enabled interactivity using slicers for time and region analysis

## Key Insights
- Sales exhibit seasonal trends with identifiable peak periods
- Technology category contributes the highest revenue
- West region outperforms other regions in sales
- A small set of products drive a large portion of revenue
- Some high-sales products have lower profit margins

## Business Impact
This dashboard enables leadership to:
- Monitor business performance at a glance
- Identify high-performing regions and categories
- Make data-driven decisions related to pricing, promotions, and inventory planning

## Tools & Technologies
- Power BI
- Power Query
- SQL (conceptual data preparation)
- Business Intelligence best practices
