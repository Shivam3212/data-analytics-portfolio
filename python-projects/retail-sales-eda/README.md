# Retail Sales Performance EDA

## Objective
Analyze retail sales data using Python to uncover revenue trends,
category performance, discount impact, and regional insights.

## Tools Used
- Python
- pandas, numpy
- matplotlib, seaborn
- Jupyter Notebook

## Key Analysis Areas
- Data quality checks
- Sales and profit trends
- Category-level performance
- Discount vs profit analysis
- Regional insights
